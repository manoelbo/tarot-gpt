Sobre o projeto

Criado por Sabine Holler e Mané Brasil, este projeto une nossa paixão por tarot e programação. O Zoltar Tarot IA é uma projeto open source, licenciada pela GNU GPL, onde exploramos a intersecção entre o misticismo do tarot e as inovações tecnológicas.

Vamos ficar muito felizes outras pessoas contribuirem com esse projeto.

Política de Privacidade do Zoltar Tarot IA

1. Introdução

Bem-vindo ao Zoltar Tarot IA. Esta política de privacidade explica quais dados coletamos de nossos usuários e como são utilizados. Nosso compromisso é garantir a segurança e a privacidade das suas informações.

2. Coleta de Dados

Os seguintes dados são coletados:

  • Número de celular: Utilizado exclusivamente para fins de identificação e comunicação com o usuário.
  • Dados de interação: Incluem a última consulta feita e avaliações de feedback. Esses dados são usados para melhorar a qualidade dos nossos serviços.
3. Política de Não Gravação

Ressaltamos que:

  • Perguntas, respostas e conversas realizadas durante o uso do serviço não são gravadas em nenhum database.
  • Não coletamos nenhum outro tipo de dado sensível ou informação pessoal além dos mencionados acima.
4. Uso Mínimo de Dados

Nossa política é utilizar o mínimo possível de dados para fornecer nossos serviços. Não armazenamos informações desnecessárias e nos comprometemos a manter a privacidade dos dados coletados.

5. Remoção de Dados

Se desejar ser removido do nosso database, por favor, envie um pedido para manebrasil123@gmail.com. Seu pedido será processado conforme as leis de proteção de dados aplicáveis.

6. Contato

Para mais informações ou dúvidas sobre nossa política de privacidade, entre em contato através do e-mail fornecido.
